# MotorPH Payroll System - Independent Build
**Author:** Magahis, Princess Jade R.

## 📋 Project Overview
This is a reconstructed Payroll Management System for MotorPH. After a structural reset, I have rebuilt the data architecture to handle 100+ employee records and automated attendance tracking.

## 📂 Repository Structure
- **data/Employee_Data.csv**: Complete master list of 100 employees including SSS, TIN, and PhilHealth.
- **data/Attendance_Logs.csv**: Daily Time Records (DTR) with varied clock-in/out times to test for lates and overtime.
- **data/MotorPH_System.java**: The core logic for processing payroll.

## 🚀 Key Improvements
- **Clean Data:** Removed redundant fields from the previous group version.
- **Scalability:** System is now optimized to handle a larger workforce (100+).
- **Standalone:** Fully independent file structure for easier deployment in NetBeans.
